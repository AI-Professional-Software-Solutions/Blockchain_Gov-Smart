# PandoraPay API

There are three types of API in order to interact with PandoraPay DAEMON

1. HTTP
   1. [ ] authentication :checked:
   2. [ ] notifications
2. HTTP RPC 
   1. [X] authentication
   2. [x] wallet
   3. [ ] notifications 
3. HTTP Websockets
   1. [X] authentication
   2. [x] wallet
   3. [X] notifications
